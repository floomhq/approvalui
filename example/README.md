# Example

Run from the repo root:

```bash
approvalui example/fixes.json example/approval.html
open example/approval.html
```

Or with the standalone script:

```bash
python3 approvalui.py fixes.json approval.html
open approval.html
```

Then tick approve or reject, add a comment, and click **Generate review**.

`screenshot.svg` is a placeholder. Replace it with your own screenshot and update the `"screenshot"` path in `fixes.json`.
